GetOrganized

A searchable, collaborative inventory system built around the structure of a home — powerful enough for any collection, hobby, or household project.

⚡ Real-Time Sync 🔗 Collaborative Sharing 🔍 Fuzzy Search 📸 Image Support

Overview

Your entire world, catalogued.

The core model is beautifully simple: a Home contains Rooms, and Rooms hold Items. That three-level hierarchy is humble on the surface — but it can model almost anything you can imagine.

🏠
Home
Your workspace
→1 → many
🚪
Room
Any space or container
→1 → many
📦
Item
Any object or entry

Full CRUD

Create, read, update, and delete Homes, Rooms, and Items. Every operation is protected, validated, and reflected in real time across all connected clients.

Not just for houses

The "room" is just a concept — it could be a tackle box, a shelf, a tote bag, a playlist, or a recipe category. The model is as flexible as your imagination.

Features

Everything you need to get organized

🏷️

Tags & Categories

Assign items to named categories and attach free-form tags. Both are scoped to your Home — filter and cross-reference your inventory any way you like.

🔍

Fuzzy Search

Search across item names, notes, categories, and tags simultaneously. Results are scored by relevance: exact → prefix → substring → fuzzy. Navigate directly to the item in its room.

📸

Multiple Images

Attach several photos to any item. Perfect for collectibles, gear identification, or documenting condition. Stored and served by the backend.

📋

Bulk Insert

Add a whole room's worth of items in one go. Ideal for the initial cataloguing sprint when you're digitizing a physical space or collection.

⚡

Real-Time Sync

Powered by Socket.io. Changes propagate instantly to all connected tabs and devices — collaborators see additions and updates the moment they happen.

🔔

Notifications

In-app notification feed for shares, invites, and collaborative changes. Unread badge updates in real time. Notifications auto-expire after 30 days.

🏘️

Multi-Home Support

Authenticate to multiple Homes from one device. Switch between a personal stash, a family home, a vacation property, or a shared workshop instantly.

👁️

Shared With Me

A dedicated dashboard showing everything others have shared with you. Switch into a shared Home, browse a shared Room, or inspect a shared Item from one place.

Collaboration

Three ways to share

Every share supports an optional edit permission — recipients can be viewers or full collaborators, your choice.

🤝 Direct Share

Share a Home, Room, or individual Item directly with a specific person using their Home token. Appears in their "Shared With Me" feed with real-time push updates when content changes.

🔗 Share Link

Generate a public URL backed by a secure base64 token. Anyone with the link can view (or edit, if permitted) — no account required. Links can be revoked at any time.

📬 Home Invite

Invite someone to join your Home as a full member. Invite links expire after 7 days. Modes: join (claim the household) or view (guest access).

Clever Touches

Nothing gets lost

Soft Delete

🗑️ The Dumpster

Deleted items and rooms aren't gone — they move to the Dumpster. Review and recover anything at any time.

  • Restore individual items or rooms with one click
  • Spring Cleaning: permanently wipe all trash at once
  • Nothing accidentally vanishes forever
Orphaned Items

🏷️ The Yard Sale

When a Room is deleted, its items don't disappear — they become orphaned and surface on the Yard Sale page.

  • Relocate items to an active room
  • Or send them to the Dumpster for later cleanup
  • Graceful, thoughtful handling of reorganization

Setup

Get running in minutes

1

Prerequisites

Node.js 18+ and a MongoDB instance (local or Atlas).

2

Install dependencies

cd backend && npm install
cd ../frontend && npm install
3

Start with PM2

npm run dev
# or: pm2 start ecosystem.config.js
4

Open the app

The backend API runs on port 5000. The frontend dev server proxies API calls automatically. Navigate to your frontend URL and your Home is created on first visit.
